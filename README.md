# Hugo Beginner Tutorial Project

This project contains a series of tutorials for beginners on how to use Hugo, a static site generator.

## Description

This repository includes:

- Markdown files with tutorials on installing Hugo, creating your first Hugo website, and understanding Hugo shortcodes.
- Images and assets used in the tutorials.
- Configuration files for Hugo.

## Installation

1. **Install Hugo**: Follow the instructions in the tutorial file `content/post/Hugo-Beginner-Tutorial—#1-Installing-Hugo.md` to install Hugo on your system.

2. **Clone the Repository**: 
   ```bash
   git clone https://github.com/your-username/xtimesai.github.io.git
   ```

3. **Navigate to the Project Directory**: 
   ```bash
   cd xtimesai.github.io
   ```

4. **Run Hugo**: 
   ```bash
   hugo server -D
   ```

## Usage

- Open your browser and navigate to `localhost:1313` to view the site locally.
- Edit the markdown files in the `content` directory to add or modify content.
